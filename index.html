Quokka

Quokka is a social media app focused on connections through quality content. I built it alongside 4 other engineers with a React/Redux frontend, Express/Node backend, and Mongo database (MongoDB). We utilized Amazon AWS S3 for picture storage and organization, Google Maps API for location data, and React Quill for rich content.

Pawdar

Pawdar is a clone of the website, petfinder.com, where users are able to find adoptable pets! I created Pawdar with a Ruby on Rails backend, React, and Redux. In addition, I utilized Amazon AWS S3 for image storage and organization.

  • Visit Pawdar

DINO DODGERS

DINO DODGERS is a simple yet engaging game for entertainment purposes. Players control two dinosaurs at the same time by moving left and right to dodge falling meteors. I created DINO DODGERS using Vanilla Javascript, HTML5 Canvas, CSS, and Webpack/Babel. Custom collision detection, event listeners, sprite sheet algorithms, and animation/stagger frames form the core of this game.
